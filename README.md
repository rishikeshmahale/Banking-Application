# Banking Application

A full-stack **MERN (MongoDB, Express, React, Node.js)** Banking Application that allows customers to manage their accounts and bankers to manage customer accounts. The application supports deposits, withdrawals, and viewing transactions with real-time updates.

---

## Features

### Customer
- Sign Up & Log In
- View current balance
- View transaction history
- Deposit and withdraw funds
- Transactions display with:
  - **Deposits** in green
  - **Withdrawals** in red

### Banker
- Sign Up & Log In
- View all customers
- View individual customer transactions

### General
- JWT authentication for secure access
- Responsive UI using Tailwind CSS
- Real-time updates using polling
- Error handling and notifications using **react-hot-toast**

---

## Tech Stack

**Frontend:** React, Redux toolkit for state management, Tailwind CSS, Axios, react-hot-toast, 
**Backend:** Node.js, Express.js, MongoDB, Mongoose  
**Authentication:** JWT  
**Deployment:** Render (Frontend and Backend)
